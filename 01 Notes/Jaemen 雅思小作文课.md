## 全局框架

### 写作结构
```
Introduction → Overview（2-3点）→ 主体段（2-3段）
```

### 核心分类（ep5）
**不要按图表外观分类，应按数据逻辑分类**：

| 分类 | 判断标准 | 代表题型 |
|------|---------|---------|
| 动态图 | X轴为连续时间 | 线形图 |
| 不连续时间 | 2-3个时间点 | 柱状图 |
| 静态图 | 无时间轴 | 饼图/柱状图 |
| 流程图 | 步骤描述 | Diagram |
| 地图题 | 空间变化 | Map |

### 表达速查表

**Introduction 句式**：

| 题型 | 模板 |
|------|------|
| 动态图 | The graph illustrates how...changed in...over... |
| 静态图 | Given are charts showing how...differ in... |
| 地图题 | Given are two maps illustrating the current layout and changes that will be introduced |
| 流程图 | The diagram illustrates the process by which...are produced |

**趋势描述**：

| 方向 | 动词 | 名词 | 高级 |
|------|------|------|------|
| 上升 | increase/rise/grow | growth/increase/rise | soar/rocket |
| 下降 | decrease/fall/decline/drop | decrease/decline/drop | plummet/plunge |
| 持平 | level off/stabilize | stability | plateau |
| 波动 | fluctuate | fluctuation | oscillate |

**程度修饰**：

| 程度 | 形容词 | 副词 |
|------|--------|------|
| 剧烈 | dramatic/sharp/significant | dramatically/sharply/significantly |
| 温和 | moderate/gradual/steady | moderately/gradually/steadily |
| 轻微 | slight/minor | slightly |
| 快速 | rapid/swift | rapidly/swiftly |

**倍数与比较**：

| 关系 | 表达 |
|------|------|
| 两倍 | twice as high/many as |
| 三倍 | three times as high/many as |
| 相等 | A equals B / with each accounting for |
| 超过 | outnumber/surpass/exceed |

**过渡与连接**：

| 功能 | 表达 |
|------|------|
| 开始描述 | commencing with / with regard to / concerning |
| 段间对比 | by contrast / similarly |
| 话题切换 | as for / when it comes to |
| 承上启下 | compared with |

## 备考原则

1. 先拆解后组装：各部分分开写，不要追求一气呵成
2. 背诵优先：前期必须背熟基本表达和框架
3. 抓大放小：整体完成度 > 细节完美
4. 时态：过去时间用过去式，无标注用一般现在时
