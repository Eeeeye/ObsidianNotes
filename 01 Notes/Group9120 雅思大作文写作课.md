## 二、核心哲学

### 雅思大作文的本质

> 以议论文为载体，陈述普遍社会观点的语言考试

**关键理解**：
- 雅思大作文 **≠ 批判性写作**，不需要标新立异、辩证深刻
- 观点必须基于 **Social Consensus（社会共识）**——无论中国人还是西方人都普遍认可的
- 雅思话题 ≠ 批判性写作话题（如堕胎、死刑）

### 写作优先级

```
立场明确 → 观点普遍合理 → 结构有序 → 语言炫技
   ↑              ↑              ↑           ↑
  前提           前提           前提       锦上添花
```

**TR 和 CC 决定分数上限**，只有切题+逻辑合理，语言炫技才有用。Off Topic 直接 5.5。

## 三、四大评分标准

| 维度 | 代码 | 7分核心要求 |
|------|------|------------|
| Task Response | TR | 普遍观点(consensus)，明确立场(position)，拓展充分 |
| Coherence & Cohesion | CC | 全文有序，逻辑衔接词+指代关系+合理分段 |
| Lexical Resources | LR | 灵活用词，less common/idiomatic items，学术风格 |
| Grammatical Range | GR | 复杂语法结构（定语从句/分词/长介词短语） |

### TR 详解

**1. 回应题干 (Address the Prompt)**：
- 8分：appropriately AND sufficiently addressed（所有关键词+修饰成分）
- 7分：main parts appropriately addressed（主要部分恰当回应）
- 6分：parts addressed but 有偏重
- 5分：incompletely addressed（漏点）

**2. 立场 (Position)**：
- Clear ≠ 极端；可以有倾向性
- 加分题(A/D)：可完全同意/反对/部分同意
- Discuss both views：可站A/B/提出新观点C/平衡
- 中立立场可接受，但优缺点分析题推荐有倾向性

**3. 结论 (Conclusion)**：
- 最好有但不是致命
- 四原则：①不同方式总结观点 ②扩大视角 ③不引入新信息 ④<50词

**4. 六种论证方法**：
1. Deductive Reasoning（演绎/三段式）
2. Inductive Reasoning（归纳法）
3. Cause and Effect Analysis（因果分析）
4. Exemplification（举例论证）
5. Compare and Contrast（对比对照）
6. Counter Argument and Rebuttal（驳论/让步拉回）

### CC 详解

**Coherence（连贯）**：
- 段内：Topic Sentence → Supporting Sentences → Concluding Sentence (optional)
- 段间：对比/并列关系也要有呼应，可用相同例子从不同角度证明

**Cohesion（衔接）** — 五类连接手段：
1. 连词 (Conjunctions)：FANBOYS — for, and, nor, but, or, yet, so
2. 副词连接词：however, moreover, additionally
3. 短语连接词：in contrast, for example, on the other hand
4. 指代关系 (Reference)：人称代词 + 指示代词 (this, that, these, those)
5. 替代关系 (Substitution)：such, so, one, other, do

**⚠️ 标点陷阱**：
- 连词可直接连两个完整句子（逗号+连词）
- 副词/短语不能连完整句子，需前句号后逗号
- `Some people claim..., but I disagree.` ✓
- `Some people claim... However, I disagree.` ✓（前句号，后逗号）

**⚠️ 常见误用**：
- `on the other hand` = 对立对比，**非并列**
- `in contrast` = 两事物差异对比
- `on the contrary` = 否定前文（"恰恰相反"）

**模糊指代**：
- 当前文有多个事物时，it/this 可能歧义
- 用 the latter, the former, the new one 替代

**分段**：推荐 4 段或 5 段，总分总结构

### LR 详解

**Risk-taking 被鼓励**：
- 花滑比喻：完美3A不如失败4A分高
- 考官鼓励尝试更难表达，即使准确度略降

**四种增加词汇灵活性的方法**：
1. General → Academic: big → substantial/grave/pressing
2. Synonyms & Antonyms: change → transformation/alteration
3. Phrasal verbs: continue → carry on; start → set off
4. Subject-specific: 环境(carbon footprint, biodiversity) / 科技(AI, cybersecurity)

**三大常见错误**：
1. 形近/音近词混淆：affect(v.)/effect(n.)、compliment/complement
2. 错误感情色彩：proud(褒)/arrogant(贬)、thrifty(褒)/stingy(贬)
3. 过度夸张 (over-exaggeration)：用词过大导致表意偏差

**idiomatic items 范例**：
- instead of → in lieu of
- in the end → in the long run
- on the whole → by and large
- as a result of → in the wake of

**学术写作风格**：
- 不用缩写 (can't → cannot，注意是一个词)
- 选正式词汇 (awesome → impressive)
- 客观语气，不要感叹号
- 不用 e.g.，用 for example

### GR 详解

**核心要求**：
- 必须结合使用：simple + compound + complex sentences
- 复杂结构：定语从句、分词结构、长介词短语
- 句子推荐长度：15~30词

**⚠️ 不要过度使用从句**：
- ✗ `This can be a solution that is feasible.`
- ✓ `This can be a feasible solution.`
- 从句多 ≠ 从句多样

**标点符号陷阱**：
- 英文**无书名号**《》，用首字母大写或斜体
- 英文**无顿号**，用逗号
- **Comma splice（逗号连接两个完整句子）** 是大忌 → 改句号或分号

## 四、审题体系（Lecture 3）

### 三步审题法

```
第一步：分辨作文类型
  ├─ 话题类（观点大作文）
  └─ 分析报告类（Two-Part）

第二步：确定核心逻辑
  ├─ 话题类 → 判断好坏对错？还是比较？
  │   ├─ Effects 好坏 → ATES
  │   └─ Facts 对错 → SFVS
  └─ 分析报告类 → ICESS

第三步：是否存在双边？
  └─ 单边 vs 双边（一个 Action/Statement vs 两个）
```

### 题型占比

| 类型 | 占比 |
|------|------|
| 单边+双边 Effects | ~55% |
| 单边+双边 Facts | ~20% |
| Two-Part Analysis | ~25% |

### 提问形式 vs 核心逻辑

提问形式不能决定一切，需结合**核心逻辑**判断：

| 提问形式 | 可能的核心逻辑 |
|----------|--------------|
| Agree/Disagree | Effects 或 Facts |
| Positive/Negative | Effects（好坏） |
| Advantages outweigh disadvantages | Effects（比较） |
| Discuss both views | Effects（两件事比较）或 Facts（两命题比较） |

### 三大审题公式

**ATES（Effects 类 — 约40%）**：
- **A**ction — 话题中施加影响的行为
- **T**arget — 行为作用的对象
- **E**ffects — 好坏结果 = 你的观点
- **S**cope — 提问方式 + 限定条件

**SFVS（Facts 类 — 约15%）**：
- **S**tatement — 提炼核心命题
- **F**acts — 搜集证明/反驳的事实
- **V**erdict — 裁决（正确/错误/部分正确）
- **S**cope — 提问方式限定

**ICESS（分析报告类 — 约25%）**：
- **I**ssue — 识别社会问题
- **C**auses — 起因（主线）
- **E**ffects — 结果（支线，可锦上添花）
- **S**olutions — 解决方案（主线）
- **S**cope — 限定条件

## 五、观点生成（Lecture 4）

### Effects 类 → Desired Effects 观点库

**核心公式**：把 Action 和题材相关的 Desired Effects 联系起来，判断是增益还是阻碍。

**教育类 Desired Effects**：
1. Academic Development（学业进步）
2. Enlightenment（世界观启蒙）
3. Career Development（职业发展）
4. Social & Communication Skills（社交沟通）
5. Physical & Mental Health（身心健康）
6. Moral Development（道德价值观）

**科技类 Desired Effects**：

个人层面：
1. Efficiency（效率）
2. Social Relationship（社交拓展）
3. Depth & Diversity in Thinking（思维深度）
4. Physical & Mental Health（健康提升）

社会宏观层面：
5. Economic Development（经济发展）
6. Social Equity & Stability（社会平等稳定）
7. Environment（环保）

**2024-2025 题材占比**：教育~25%，科技~15%，社会问题~30%

### Facts 类 → Common Facts / Counter Facts

- 证明命题正确：列举社会普遍事实
- 证明命题错误：举反例即可（不需要像证明正确那样严谨）
- 逻辑方向：Effects 由内向外，Facts 由外向内

## 六、写作结构（Lecture 5）

### 三种结构

| 结构 | 适用场景 | 特点 |
|------|---------|------|
| **单边 (One-sided)** | 判断好坏、Facts对错 | 全正面或全反面 |
| **双边 (Two-sided)** | 比较好坏、双边讨论 | 三段：让步/平衡/拉回 |
| **Two-Part** | 分析报告类 | 按 Cause→Effect→Solution 顺序 |

### 双边结构的三种姿态

1. **平衡 (Balanced)**: A有道理，B也有道理，各有千秋
2. **让步不拉回 (Concession)**: A有道理，但B更重要，Overall B
3. **让步拉回 (Concession + Rebuttal)**: A有道理，但是A可以被反驳 → Overall B

## 七、论证体系（Lecture 6-7）

### CAE 论证法（Effects 类）

```
Condition → Action → Effect
 条件前提 → 行为机制 → 结果效应
```

**基础版 (~69词, 6.5-7分)**：
- Topic Sentence 声明观点
- Condition：目标身上什么特质/需求让行为能产生作用
- Action：行为如何一步步改善问题
- Effect：总结效果，呼应观点

**高分进阶版 (7-7.5分)**：添加定语从句、分词结构、长介词短语

**变形技巧**：
- **双重AE**：省略 Condition，两个 Action→Effect（8分策略）
- **省略Effect做正反对比**：先证明不这么做会怎样
- **加入具体例子**：在论证后附加例证，增加灵动性

### CAF 论证法（Facts 类）

```
Condition → Action → Fact
 条件前提 → 行为机制 → 普遍事实
```

不同于 CAE 的 Effect（推导结果），CAF 用 Fact（已有事实）支撑命题。

可结合：反证 + 让步拉回 + 例证

### 让步拉回 (Concession and Rebuttal)

**本质**："欲扬先抑"——先提反方观点，再反驳，加深己方立场

**关键原则**：
- 让步和拉回必须在**同一观点维度**下进行
- 不能让步"健康"，拉回"效率"——完全不同维度

**模板结构**：
```
Still, some might claim that [反方观点].
However, [反驳理由].
Therefore, [己方结论].
```

**高分技巧**：在开头段预埋伏笔
```
It cannot be denied that [反方现象]. However, [全文立场].
→ 开头埋下让步拉回的伏笔，后面呼应
→ 同时提升 TR 和 CC 分数
```

**写作工具优先级**：
```
Reasoning (CAE/CAF) > Exemplification > Concession & Rebuttal > Opening/Ending
```

## 八、开头与结尾（Lecture 9）

### 核心原则

> 作文主体 Body Paragraphs 最重要，开头是次要的，结尾是锦上添花——写得好才要，没必要硬凑。

- 不少7.5甚至8分作文是没有结尾段落的
- 开头就算很简单，也只是"不出彩"，不会直接扣分
- 关键：**先写对，再写好**

### 开头段落结构

```
Background Information + Common Points + Position
```

**单边结构开头范例**（简洁版）：
```
It is debated that [话题]. As [观点概述] are concerned, I disagree with this view.
```

**双边结构开头范例**：
```
It is debated among people if [话题]. In my opinion, as far as [正面观点] and [负面观点] are concerned, such experience is overall advantageous but not without drawbacks.
```

**让步拉回开头（进阶）**：
```
It cannot be denied that [反方现象]. However, as far as [正面观点] are concerned, [全文立场].
```

### 结尾段落

公式：**Summary + Sublimation**（总结前文 + 延伸/升华）

范例：
```
In conclusion, the use of computer has permeated every corner of modern jobs, and a mastery of excellent computer skills is necessary to a successful personal career in almost any industry.
```

## 九、分析报告类专题（Lecture 10）

### Causes 两大来源

1. **内在原因 (Intrinsic)**：普遍存在，不限于时代/地域
2. **外部社会因素 (External Social)**：经济、科技、文化、生活方式

当今世界变化最大推手 = **经济 + 科技发展 + 文化变革**

### Solutions 两大原则

1. 从根源(Causes)出发解决
2. 社会各方协同：国家/政府 → 社会组织/企业 → 个人

### 结构模板

- 开头：`It is a common phenomenon today that [issue]... which is most likely caused by a combination of intrinsic and social factors.`
- Causes段：`First of all, in terms of the causes... What makes it worse is that...`
- Effects段（支线）：`Such phenomenon is a dire issue...`
- Solutions段：`As for solutions, [parties] shall... Meanwhile...`
- 结尾：可选

## 十、常见错误与陷阱

| 陷阱 | 说明 |
|------|------|
| 审题一上来就"意会" | 雅思是标准化考试，有固定套路，不能凭经验解读 |
| 忽略 Scope 限定 | 如"for study"限定了讨论范围，不能泛泛而谈 |
| Effects/Facts 混淆 | 看到 negative effects 就往 Effects 走，实际可能是 Facts 类 |
| 用好处否定坏处 | 列举3个好处不能证明"没有坏处"这个命题是错的 |
| 过度换词 (paraphrase) | 为了换词偏离原意反而不如原词 |
| Comma splice | 逗号连接两个完整句子 → 改为句号或分号 |
| 模糊指代 | 前文多事物时 it/this 指代不明 |
| on the other hand 误用 | 当作并列使用，实际是对立对比 |
| Off Topic | 无论语言多华丽，偏题直接上限 5.5 |
| 硬凑复杂从句 | 从句多 ≠ 多样，简洁就好时不需从句 |

## 十一、实用建议

**字数与分数参考**：
- 6~6.5分：250~300词
- 6.5~7分：275~350词
- 7分以上：一般 >300词，甚至400+

**日常练习方法**：
- 先用记事本写作（无拼写检查），再复制到Word检查
- 机考有实时字数统计但不会标注错误
- 考前应急：直接跳至 Lecture 10 Composition，再回头找工具

**推荐书籍**：
- 《纯正英式短语和句型一千条》
- English Collocations in Use (Cambridge)
