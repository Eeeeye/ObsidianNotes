| conservatory | 温室；玻璃暖房 | /kənˈsɜːvətri/ | 建筑用语，指玻璃搭建的阳光房/温室；⚠️ 别和 `conservation`（保护）、`conservative`（保守的）混淆 |

| sculpture | 雕塑；雕刻品 | /ˈskʌlptʃə(r)/ | 注意和 `sculptor`（雕塑家）区分；`bronze sculpture` = 青铜雕塑；`garden sculpture` = 花园雕塑 |
| intention | 意图；目的 | /ɪnˈtenʃən/ | ⚠️ 和 `extension`（扩建）发音接近——在建筑语境中常见的是 `extension`（房屋扩建），确认原文语境 |
| allies | 盟友；同盟国 | /ˈælaɪz/ | 单数 `ally`；`allies` 复数；动词 `ally with` = 与…结盟；⚠️ 也和 `alleys`（小巷/小径）发音相同，庄园语境下也可能是 alley |
| servant | 仆人；佣人 | /ˈsɜːvənt/ | `servants' quarters` = 仆人住宿区；`civil servant` = 公务员；⚠️ 别和 `server`（服务器）混淆 |
| tractor | 拖拉机 | /ˈtræktə(r)/ | 常见于农场/庄园语境；`tractor shed` = 拖拉机棚；`farm tractor` = 农用拖拉机 |
| dominate | 主宰；支配；占据显著位置 | /ˈdɒmɪneɪt/ | 原文语境："building that dominates the farm"（在农场景观中最显眼的建筑）；作视觉义时 = 占据最突出位置；`dominate the skyline` = 主宰天际线 |
| plough the earth | 犁地；翻土 | /plaʊ ði ɜːθ/ | `plough` = 犁（n./v.）；已在 [[剑10 Test4 听力复盘]] 的 `plough ahead` 中记录过比喻义；此为本义：用犁翻耕土地 |
| sow seed | 播种 | /səʊ siːd/ | ⚠️ `sow` /səʊ/ 和 `sew`（缝纫）同音！`sow` 作动词读 /səʊ/（播种），作名词读 /saʊ/（母猪），同形异音；`sow seeds` = 播种（literal）也可比喻"播下…的种子" |
| groom | 刷洗；梳洗（马/驴）；新郎 | /ɡruːm/ | 原文语境："groom the donkeys and horses"（给驴和马梳洗）；作名词 = 新郎；作动词还可表示"培养/训练"（`groom a successor`） |
| blanket | 毯子；毛毯 | /ˈblæŋkɪt/ | 原文语境：`horse blanket` = 马毯（给马保暖用）；⚠️ 拼写注意：不是 `blankey`，发音像 /blæŋki/ 但结尾是 -et 不是 -ey |
| shed | 棚；小屋 | /ʃed/ | `tractor shed` = 拖拉机棚；`garden shed` = 花园工具棚；⚠️ 也作动词"脱落/流下"（`shed tears/light`），意思完全不同 |
| brick | 砖；砖块 | /brɪk/ | `brick building` = 砖楼；`brick wall` = 砖墙；常见建筑材料和形容词 |
| carriage | 马车；车厢 | /ˈkærɪdʒ/ | `horse-drawn carriage` = 马车；`carriage house` = 马车房；⚠️ 别和 `marriage`（婚姻）混淆，发音接近 |
| breed | 品种（n.）；繁殖（v.） | /briːd/ | 已在 [[剑17 Test1 听力复盘]] P3 的 `lowland breed` / `bred` / `reared for` 中记录过；再次出现说明是高频农业词 |
